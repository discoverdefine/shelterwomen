How do I get started?

In order to run the application for the first time, or to ensure that you have the most up-to-date information, you will need to connect to the Internet (a WiFi connection is recommended to avoid additional data fees) and SYNC with the online database.

Once the syncing process is complete, you should be up to date and ready to take your app with you when you visit clients.

From this point on, you will only need to connect to the Internet when you wish to grab the latest data.

Updating the directory may take some time so please be patient while the process runs its course.

How do I use this app?

In addition to an "About" tab which provides some background information regarding both the Children's Aid Society and Violence Against Women organizations, The Shelter for Women app includes two important functions that you can use when working with your clients.

Directory

An up to date list of shelters - including addresses, important contact numbers, and web sites where applicable. Maps help to identify the most suitable locations for each case.

Tours

Put your clients at ease by showing them around actual shelters being used today. Photos and panoramic views can help to convey the warm and welcoming atmosphere while not compromising security or anonymity.

Do I need the Internet?

An Internet connection is not required to run the application for everyday use. In order to ensure that you are working with the most up-to-date information available, it is recommended that you connect to the Internet (a WiFi connection is recommended to avoid additional data fees) and then SYNC (See "How do I get the latest data?") with the online database.
